Welcome to Bootleg
====================================

Bootleg_ is a named entity disambiguation (NED) system that links mentions in text to entities and produces contextual entity embeddings.

Bootleg is still *actively under development*, so feedback and
contributions are welcome. Submit bugs in the Issues_ section or feel free to
submit your contributions as a pull request.

.. _Issues: https://github.com/HazyResearch/bootleg/issues/
.. _Bootleg: https://github.com/HazyResearch/bootleg
